Toggle Component Interactive

A custom toggle element styled entirely with :state(checked) and :state(unchecked). Click to toggle.
States: unchecked
/* CSS */ custom-toggle:state(checked) .track { background: #6c8cff; } custom-toggle:state(unchecked) .track { background: #3a4260; } // JavaScript this._internals = this.attachInternals(); this._internals.states.add('unchecked'); // on click: this._internals.states.delete('unchecked'); this._internals.states.add('checked');
